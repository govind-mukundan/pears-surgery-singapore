# Cross-Border Asset Checklist (Singapore + India)

Use this as your working checklist and update status as you go.

## A. Identity and Core Records

- [ ] Passport and primary ID copies prepared.
- [ ] Current residential address proof prepared.
- [ ] Tax identifiers recorded for both jurisdictions (as applicable).
- [ ] Emergency contacts list updated.

## B. Asset Inventory (Financial Only)

For each account/holding, capture:

- [ ] Institution name.
- [ ] Account type (savings/fixed deposit/brokerage/demat/trading).
- [ ] Jurisdiction (Singapore or India).
- [ ] Holder type (single/joint).
- [ ] Nominee/beneficiary on file.
- [ ] Approximate balance/value.
- [ ] Relationship manager or support contact.

### Suggested Table Format

| Jurisdiction | Institution | Account Type | Identifier (masked) | Nominee | Approx Value | Notes |
|---|---|---|---|---|---|---|
| Singapore | Example Bank | Savings | XXX1234 | Yes/No | SGD ... | |
| India | Example Broker | Demat/Trading | XXX9876 | Yes/No | INR ... | |

## C. Beneficiary and Executor Alignment

- [ ] Beneficiary names in will match legal IDs.
- [ ] Account nominations reviewed and aligned with will strategy.
- [ ] Primary and backup executors confirmed.
- [ ] Cross-border contact details shared with executor.

## D. Will Execution Controls

- [ ] Jurisdiction strategy selected (single will vs two coordinated wills).
- [ ] Draft reviewed by Singapore counsel.
- [ ] Draft reviewed by India counsel.
- [ ] Witness eligibility confirmed.
- [ ] Signed originals safely stored.

## E. LPA Controls (Singapore)

- [ ] Donee and replacement donee selected.
- [ ] Scope of powers decided (including financial operations).
- [ ] Certification appointment completed.
- [ ] LPA registration submitted/completed.
- [ ] Donee briefed on practical operations.

## F. Access and Continuity (Without Storing Passwords in Plain Text)

- [ ] Password manager emergency access plan in place.
- [ ] 2FA recovery method documented and secured.
- [ ] Device unlock contingency plan prepared.
- [ ] Critical bill autopay list documented.

## G. Pre-Surgery Final Check

- [ ] Executor has lawyer contact and document location.
- [ ] Donee knows LPA status and where records are stored.
- [ ] At least one trusted person knows emergency file location.
- [ ] Document versions/date updated.

## H. Annual Review Trigger

Review this checklist every 12 months or immediately after:

- Opening/closing accounts.
- Beneficiary changes.
- Major portfolio changes.
- Jurisdiction/residency changes.
